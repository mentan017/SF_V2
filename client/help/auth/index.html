How to reset a password

The easiest way of reseting password is to directly modify the MongoDB database. In both cases, we will need access to the database, ideally through MongoDB Compass

Method 1: Removing the password and activating the user

In order to use this method, we need access to an account/user with the permission CanManageAllUsers set to true, we will call this user "admin".
First, we delete the password from the database, as shown here using MongoDB Compass:

Then the admin goes to the user page of the user requesting a password reset and he activates the user. This will provide the admin with a new password for the user.
